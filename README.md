Среднее арифметическое элементов списка 'c', расположенных на четных местах. Среднее геометрическое элементов списка 'c', расположенных на нечетных местах. Максимальный и минимальный элементы. b = []

a.append(3.4)
# Добавить 2 вещественных числа (8.7, 1.3) в список 'a' с помощью extend

b.extend([8.7, 11.3])
